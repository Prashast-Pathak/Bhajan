Bhagavad Gita 2.17

Programmatic spiritual reference page

Sanskrit: अविनाशि तु तद्विद्धि येन सर्वमिदं ततम्। विनाशमव्ययस्यास्य न कश्चित्कर्तुमर्हति॥

Meaning: Know that to be indestructible by which all this is pervaded. No one can cause the destruction of the imperishable soul.
